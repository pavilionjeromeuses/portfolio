Standardized File Naming System

ByteOrder Systems - Effective as of Q3 2023

Naming Convention

Format: [ProjectCode]_[Department]_[Date]_[Description].[Ext]

  • Project Code: PRJ-XXXX (4-digit number)
  • Department: DEV, MKTG, HR, FIN, OPS
  • Date: YYYYMMDD
  • Description: Short descriptive text (kebab-case)
  • Extension: Standard file extension
